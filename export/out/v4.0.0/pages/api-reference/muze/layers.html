BaseLayer

This is an abstract class which needs to be inherited by other concrete layers. It has all the common methods implemented required for all layers like merging default configuration with user configuration, applying interaction styles, transforming data to stack or group and disposing layer. All these functionality can be overridden by the layer classes.

static formalName

Syntax: formalName()

Returns the unique name of the layer. This method should be implemented by all the layers inheriting this layer. To create a new layer, whatever name is returned by the formalName() is used as mark field when that layer is fed to canvas.layers()

Returns

string: Unique name of the layer

static defaultConfig

Returns the default configuration of layer. This method needs to be implemented by all the layers.

Returns

object: Default configuration of the layer.

data

Gets or sets a datamodel instance in the layer. This method is mostly used when you need to change the data of a specific layer dynamically. For example, on interaction, if you want to change the data of layer which shows an average value of a field, then you need to dynamically set the datamodel instance which contains the aggregated values to the layer.

When used as a setter,

Syntax: data(dataModel)
parameters:
		- dataModel:
				required: true
				type: DataModel
				description: Instance of datamodel which is used by layer to create the
					visualization.

Returns

Instance of layer.

When used as a getter,

Syntax: data()

Returns

Datamodel Instance of datamodel.

config

Gets or sets the configuration of the layer. When configuration is passed, then it is merged with the default configuration of the layer and set to the layer config.

When used as a setter,

Syntax: config(config)
parameters:
		- config:
				required: true
				type: CommonLayerConfig*
				description: Layer configuration.

*CommonLayerConfig

parameters:
		- className:
				required: false
				type: string
				description: Custom class name applied to the plot svg group.
		- transform:
				required: false
				type: Transform
				description: Transform configuration
		- transition:
				required: false
				type: Transition
		- interaction:
				required: false
				type: InteractionEffectConfig
				description: Interaction effect information.

*Transform

parameters:
		- type:
				required: false
				type: string
				default: identity
				description: Transform type of the plot.

*Transition

parameters:
		- effect:
				required: false
				type: string
				default: cubic
				description: Easing effect of transition.
		- duration:
				required: false
				type: number
				default: 1000
				description: Duration of transition.

*InteractionEffectConfig

This is an object with key as the name of the interaction effect and value is an object containing the styles and properties which will get applied to each plot elements when any interaction has occurred.

type: object
parameters:
	- [effect]:
			required: true
			type: object
			description: Name of the effect will be registered in interaction config
				and the effect styles will get applied when applyInteractionStyle method
				will be called with the effect name and unique ids of the elements.
			parameters:
				- style:
						required: true
						type: object
						description: Styles which will be applied to plot elements.
				- strokePosition:
						required: false
						type: string
						default: center 
						description: Position of the border.

				the domain of the x-axis will be union of both x and x0 fields.
			parameters:
				field:
					required: true
					type: string
					description: Name of x0 field
				value: EncodingValue*
					required: false
					type: number | function
					description: Returns the x0 value of the bar in pixels.
		- y0:
			required: true
			type: string | object
			description: If y0 field is present, then the height of the bar will be
				calculated from the difference between y and y0 coordinates. Also,
				the domain of the y-axis will be union of both y and y0 fields.
			parameters:
				field:
					required: true
					type: string
					description: Name of y0 field
				value: EncodingValue*
					required: false
					type: number | function
					description: Returns the y0 value of the bar in pixels.
    • Transform(from CommonLayerConfig)
      parameters:
		- type:
				required: false
				type: string
				default: identity // Supported values are identity, group and stack.
				description: Transform type of the plot. Changing this type will
					convert a bar chart to stacked or grouped.

config

Gets or sets the configuration of layer.

Syntax: config(config)

			required: true
			type: object
			description: Line layer configuration
			parameters:
				- connectNullData:

						description: If true, then connects the null data points in the line path
							else it creates a gap across the missing data points.
				- nullDataLineStyle:
						required: false
						type: Object
						default: {}
						description: If connectNullData is true, this is used as a block of css with 
						css like key value pair.

						required: false
						type: string
						default: linear // Supported values are: linear, catmullRom, step.
						description: Changes the shape of the line path. Setting it to catmullRom
							will create a curved line path and step will create a path consisting of
							alternating horizontal and vertical lines.

Encoding Value

Syntax: value: (d, i, dataArr, layerInst)

This is a function which takes value of encoding and returns a new value. For example, if you want to shift the x position of the text by 10 pixels.

parameters:
	- d:
		 required: true
		 type: DataObject*
		 description: Data object of the plot element.
	- i:
		 required: true
		 type: number
		 description: Index of the plot element.
	- dataArr:
		 required: true
		 type: Array
		 description: Array of all the data objects.
	- layerInst:
			required: true
			type: BaseLayer
			description: Instance of the layer.

*DataObject

It is an object which may have the following properties.

parameters:
	- x:
		type: number
		description: x-coordinate of the plot element.
	- y:
		type: number
		description: y-coordinate of the plot element.
	- width:
		type: number
		description: width of the plot element. Present in bar layer only.
	- height:
		type: number
		description: height of the plot element. Present in bar layer only.
	- angle:
		type: number
		description: Angle value of the plot element in radians. Present in arc layer
			and other layers which supports polar coordinates like tick and text.
	- color:
		type: string
		description: color string of the plot element.
	- shape:
		type: string.
		description: Shape of plot element. Present in point layer only.
	- size:
		type: number
		description: Size value of the plot element. Present in point layer only.

Usage

const env = muze();
const canvas = env.canvas();

canvas.layers([{
	// Bar layer definition
	mark: 'text',
	encoding: {
		x: {
			value: (d, i, dataArr, layerInst) => {
				return d.x + 10;
			})
		}
	}
}]);

Returns

string|number: Encoding value

layerFactory

Creates layer instances and registers composite layer definitions. Layer instances are created internally by canvas using layerFactory.

To access layerFactory

const layerFactory = muze.layerFactory;

composeLayers

Syntax: composeLayers(layerName, definition)

Registers composite layer definition in the layer factory.

parameters:
		- layerName:
				required: true
				type: string
				description: Name of the composite layer.
		- definition:
				required: true
				type: Array
				description: Definitions of the layers.

Returns

layerFactory

Usage

const layerFactory = muze.layerFactory;
layerFactory.composeLayers('tickPoint', [{
    name: "errorTick",
    mark: "tick",
    encoding: {
        y0: "tickPoint.encoding.lowerLimit",
        y: "tickPoint.encoding.upperLimit",
        x: "tickPoint.encoding.x",
        color: {
          value: () => '#4c8579'
        }
    },
    transform: {
        type: "identity"
    }
}, {
    name: "errorPoint",
    mark: "point",
    encoding: {
        y: "tickPoint.encoding.center",
        x: "tickPoint.encoding.x",
        color: {
          value: () => 'red'
        }
    },
    transform: {
        type: "identity"
    }
}]);
