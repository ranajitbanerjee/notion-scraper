Just like scales and axes ,Muze provides ability to encode your information into different color, shape or size . Like different categorial values can be broken into multiple colors or shape where as a quantitative value can be encoded into different size ranges.

Muze supports following types of legend :

  • Discreet Legends
    • Color Legend
    • Shape Legend
    • Size Legend
  • Continuous Legends
    • Gradient Legend
    • Step Legend

Adding a Color Legend.

Color Legends can be used to show both categorical data and quantitative values.

Below are ways you can configure a color legend.

  • color(fieldName)
    • If fieldName is dimension , then a discreet legend is created.
      ...
		.color('Cylinders')
...
    • If fieldName is measure , then a gradient legend is created.
      ...
		.color('Horsepower')
...

      You can provide following additional configuration also

      ...
.color({
	field : 'fieldName',
	step : true | false,
	stops : 10
})
...

      You can provide number stops you want in your legend by providing the same in stops attribute, also it accepts an array of integers if you want specify custom stops.

      ...
		.color({
				field : 'Horsepower',
				stops : 10,
			// stops : [20,30,50,120]
		})
...

      When step is provided then a Step Legend is formed

      ...
		.color({
				field : 'Horsepower',
				step : true,
				stops : 10,
			//stops : [20,30,50,120] 
		})
...

Adding a Size Legend

Size legends are mostly used to encode measure.

Below are the ways a Size Legend can be configured.

  • size('fieldName') 
    ...
		.size('Horsepower')
...

  • You can give number of stops or a custom range
    ...
		.size({
				field : 'Horsepower',
				stops : 10,
			//stops : [20,30,50,120] 
		})
...
