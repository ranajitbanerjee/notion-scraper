Environment, like the name suggest, is similar to the concept of closure in ECMAScript. Everytime a Canvas is created from environment, it gets the configuration from environment itself. All the getter setter methods in environment except canvas are also available in Canvas component. For example, if you want different width and height for each canvas, you can set it on canvas itself.

canvas

Syntax: canvas()

Returns an instance of Canvas which can be used to create a visualization. All canvases created from the same environment will get all the configurations passed in that environment.

Usage :-

const env = muze(); // Creates an environment

const canvas = env.canvas(); // Returns an instance of canvas

Returns

Canvas Instance of canvas

data

Gets or sets a Datamodel instance in the environment.

When used as a setter,

Syntax: data(data)
parameters:
  - data:
    	required: true
			type: DataModel

Returns

Instance of environment.

Usage :-

const dm = new DataModel(data, schema);
const env = muze(); // Creates an environment
env.data(dm); // Sets the datamodel instance

When used as a getter,

Syntax: data()

Usage :-

const dm = new DataModel(data, schema);
const env = muze(); // Creates an environment
env.data(dm); // Sets the datamodel instance
const returnDm = env.data(); // Gets the datamodel instance from the environment.

Returns

Instance of Datamodel

minUnitWidth

Gets or sets the minimum unit width of the chart. When a faceted visualization is created using muze, it consists of multiple charts which are called visual units. minUnitWidth is the minimum horizontal space each chart is allocated during space allocation.

When used as a setter,

Syntax: minUnitWidth(mintUnitWidth)
parameters
	- minUnitWidth:
    	required: true
			type: number
			default: 150
			description: Minimum width of each visual unit.

Returns

Instance of environment.

Usage :-

const env = muze(); // Creates an environment
env.minUnitWidth(200); // Sets the minimum unit width

Example

When used as a getter,

Syntax: mintUnitWidth()
const env = muze(); // Creates an environment
env.minUnitWidth(200); // Sets the minimum unit width
const minWidth = env.minUnitWidth(); // Gets the minimum unit width.

Returns

Minimum unit width of chart.

config

Gets or sets the configuration of canvases. Each canvas created from the same environment will share the configuration passed to that environment. When configuration is passed, then it is merged with the default configuration and set to canvas.

When used as a setter,

Syntax: config(config)
parameters:
		- config:
				required: true
				type: object
				default: {}
				description: Canvas configuration.
				parameters:
						- axes: Axis Configuration
								required: false
								type: object
								default: {}
								description: Configuration of axes.
						- legend: Legend Configuration
								required: false
								type: object
								default: {}
								description: Configuration of legend.
						- sort:
								required: false
								type: object
								default: {}
								description: Data sort configuration.
						- invalidValues:
								required: false
								type: object
								default: {}
								description: Configuration for formatting invalid values.
						- interaction: Interaction Configuration
								required: false
								type: object
								default: {}
								description: Interaction configuration
						- showHeaders:
								required: false
								type: boolean
								default: false
								description: Toggles the visibility of facet headers
						- autoGroupBy:
								required: false
								type: object
								default: {
										disabled: false
								}
								description: Muze internally performs a groupby on the datamodel with
									all the fields present in the chart. If autoGroupBy is disabled, then
									muze does not perform any groupby operation on the datamodel.
								parameters:
										- disabled:
												required: false
												type: boolean
												default: false
												description: Disables the internal groupby of datamodel.

References :-

* Legend Configuration

* Axis Configuration

* Interaction Configuration

Returns

Instance of environment.

Usage :-

const env = muze(); // Creates an environment
const canvas = env.canvas(); // Creates an instance of canvas.

canvas.config({
	showHeaders: true,
	axes: {
			x: {
					show: false
			}
	}
});

Example

When used as a getter,

Syntax: config()

Returns

Configuration of canvases.
