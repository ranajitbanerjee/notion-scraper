BaseLayer

This is an abstract class which needs to be inherited by other concrete layers. It has all the common methods implemented required for all layers like merging default configuration with user configuration, applying interaction styles, transforming data to stack or group and disposing layer. All these functionality can be overridden by the layer classes.

static formalName

Syntax: formalName()

Returns the unique name of the layer. This method should be implemented by all the layers inheriting this layer. This name needs be passed in mark property in layer definition of canvas.layers() method.

Returns

string: Unique name of the layer

static defaultConfig

Returns the default configuration of layer. This method needs to be implemented by all the layers.

Returns

object: Default configuration of the layer.

data

Gets or sets a datamodel instance in the layer. This method is mostly used when you need to change the data of a specific layer dynamically. For example, on interaction, if you want to change the data of layer which shows an average value of a field, then you need to dynamically set the datamodel instance which contains the aggregated values to the layer.

When used as a setter,

Syntax: data(dataModel)
parameters:
		- dataModel:
				required: true
				type: DataModel
				description: Instance of datamodel which is used by layer to create the
					visualization.

Returns

Instance of layer.

When used as a getter,

Syntax: data()

Returns

Datamodel Instance of datamodel.

config

Gets or sets the configuration of the layer. When configuration is passed, then it is merged with the default configuration of the layer and set to the layer config.

When used as a setter,

Syntax: config(config)
parameters:
		- config:
				required: true
				type: object
				description: Layer configuration.

When used as a getter,

Syntax: config()

Returns

object: Configuration of layer.

remove

Syntax: remove()

Disposes the layer elements and objects.

Returns

Instance of layer.

applyInteractionStyles

Applies interaction styles to the layer.

parameters:
	- config:
			required: true
			type: BarConfig
			description: Bar layer configuration.

  • BarConfig
    type: object
parameters:
		- innerPadding:
				required: false
				type: number
				default: 0.1
				description: Padding between two bars when grouped bar is plotted. In
					case of stacked and normal bar charts, padding between bars is calculated
					from the axis padding property.
    • Encoding
      parameters:
		- x0:
			required: true
			type: string | object
			description: If x0 field is present, then the width of the bar will be
				calculated from the difference between x and x0 coordinates. Also,
				the domain of the x-axis will be union of both x and x0 fields.
			parameters:
				field:
					required: true
					type: string
					description: Name of x0 field
				value: EncodingValue
					required: false
					type: number | function
					description: Returns the x0 value of the bar in pixels.
		- y0:
			required: true
			type: string | object
			description: If y0 field is present, then the height of the bar will be
				calculated from the difference between y and y0 coordinates. Also,
				the domain of the y-axis will be union of both y and y0 fields.
			parameters:
				field:
					required: true
					type: string
					description: Name of y0 field
				value: EncodingValue
					required: false
					type: number | function
					description: Returns the y0 value of the bar in pixels.
    • Transform
      parameters:
		- type:
				required: false
				type: string
				default: identity // Supported values are identity, group and stack.
				description: Transform type of the plot. Changing this type will
					convert a bar chart to stacked or grouped.

Encoding Value

Syntax: value: (d, i, dataArr, layerInst)

This is a function which takes value of encoding and returns a new value. For example, if you want to shift the x position of the text by 10 pixels.

parameters:
	- d:
		 required: true
		 type: object
		 description: Data object of the plot element.
	- i:
		 required: true
		 type: number
		 description: Index of the plot element.
	- dataArr:
		 required: true
		 type: Array
		 description: Array of all the data objects.
	- layerInst
			required: true
			type: BaseLayer
			description: Instance of the layer.

Usage

const env = muze();
const canvas = env.canvas();

canvas.layers([{
	// Bar layer definition
	mark: 'text',
	encoding: {
		x: {
			value: (d, i, dataArr, layerInst) => {
				return d.x + 10;
			})
		}
	}
}]);

Returns

string|number: Encoding value

Encoding Transform

Syntax: encodingTransform: (points, layerInst, dependencies)

Takes an array of points generated by the layer and transforms the values of encodings. This method is needed when you want to remove collision between labels in text layer because text layer does not manage the collision internally.

parameters:
	- points:
			required: true
			type: Array<EncodingValues*>
			description: An array of points containing the values of encodings like x, y,
				color, shape and size.
	- layerInst:
			required: true
			type: BaseLayer
			description: Instance of layer
	- dependencies:
			required: true
			type: object
			description: Dependencies needed to calculate size of text.
			parameters:
				- smartLabel:
						required: true
						type: SmartLabel*
						description: Instance of smartlabel for calculating dimensions of text

*EncodingValues
