What is a Crosstab ?

Crosstab/Pivot/Matrix view is a type of visualization in data analysis. Cross tabs allow you to create powerful views of your data summarized in a grid, both in horizontal and vertical columns. You can also create multi-level cross tabs specifying multiple fields to the column, row and data axis.

Cross tabs come in handy when you want to summarize and extract the significance from a large, detailed data set. You can even use conditional formatting options to identify key performance indicators that matter the most to your organization.

You can set dynamic filters, group by categories, drill down to view the underlying data, apply functions on columns, and do much more.
